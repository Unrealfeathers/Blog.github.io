Java入门-1-基础语法

发表于 2023-08-26 分类于 Java

学院：人工智能与数据科学学院
学级：2021级
专业：物联网工程

1. 注释

  • 单行注释 //注释信息
  • 多行注释 /* 注释信息 */
  • 文档注释 /** 注释信息 */

2. 关键字

  • class：用于创建/定义一个类

3. 字面量类型

  • 整数类型
  • 小数类型
  • 字符串类型-“”
  • 字符类型-‘’
  • 布尔类型-true/false
  • 空类型-null（无法被打印）

4. 特殊字符

  • \t（制表符）：在打印的时候，把前面字符串的长度补齐到8，或者8的整数倍。最少补一个空格，最多补8个空格。

5. 数据类型

  • 整数型-long/int/short/byte(long类型变量，数据值后需要加L)
  • 小数型-double/float(float类型变量，后缀F)
  • 字符型-char
  • 布尔型-boolean
    注：变量使用前需要赋值

6. 标识符命名规则

  • 由 数字、字母、下划线、$ 构成、
  • 不能以数字开头
  • 不能是关键字
  • 区分大小写
