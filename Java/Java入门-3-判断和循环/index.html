Java入门-3-判断和循环

发表于 2023-08-26 分类于 Java

学院：人工智能与数据科学学院
学级：2021级
专业：物联网工程

判断

  1. if语句：

    关系表达式为true，执行语句体；
    关系表达式为false，不执行语句体；

    关系表达式为true，执行语句体_1；
    关系表达式为false，执行语句体_2；

    关系表达式_1为true，执行语句体_1；
    关系表达式_1为false，关系表达式_2为true，执行语句体_2；
    关系表达式_1为false，关系表达式_2为false，执行语句体_3；

  2. switch语句：

    首先计算表达式的值，将其依次与case后的值作比较，有对应的值将执行相应的语句，遇到break结束。无值匹配，则执行default中的语句体。

循环

  1. for语句：

    其中 int i=0 只执行一次，每次循环先判断 i < number为真则继续执行，为假则停止循环。每次循环都会执行一次 i++ 。

  2. while语句：

    条件判断语句为true，循环继续；
    条件判断语句为false，循环终止；

    首先执行 CodeBlock; ，条件判断语句为true，循环继续；条件判断语句为false，循环终止。

  3. 相关关键字：

  • continue:跳过本次循环。继续下次循环
  • break:结束本层循环
