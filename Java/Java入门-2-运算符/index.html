Java入门-2-运算符

发表于 2023-08-26 分类于 Java

学院：人工智能与数据科学学院
学级：2021级
专业：物联网工程

1. 算术运算符

符号 作用
+ 加
- 减
* 乘
/ 除
% 取模/取余
++ 变量+1
– 变量-1

注：数据类型转换

  • 隐式转换：取值范围小->取值范围大（自动）
    • byte/short/char 运算时，先提升为int，再进行运算
  • 强制数据转换：取值范围大->取值范围小（手动）

2. 赋值运算符

符号 作用 说明
= 赋值 int a=10; 将10赋给变量a
+= 加后赋值 a+=b 等于 a=a+b
-= 减后赋值 a-=b 等于 a=a-b
*= 乘后赋值 a*=b 等于 a=a*b
/= 除后赋值 a/=b 等于 a=a/b
%= 取余后赋值 a%=b 等于 a=a%b

3. 关系运算符

符号 说明
== 判断是否等于
！= 判断是否不等于
> 判断是否大于
>= 判断是否大于等于
< 判断是否小于
<= 判断是否小于等于

4. 逻辑运算符

符号 作用 说明
& 逻辑与 两边为真，结果为真
| 逻辑或 一边为真，结果为真
^ 逻辑异或 相同为真，不同为假
！ 逻辑非 取反
&& 短路与 结果与&相同，但是有短路效果
|| 短路或 结果与|相同，但是有短路效果

注：
“&&”-第一个为假，不判断第二个，直接输出false；
“||”-第一个为真，不判断第二个，直接输出true；
“&|”-无论左边true or false，右边均执行

5. 三元表达式

  • DataType a = number_1 > number_2 ? number_1 : number_2;
  • 如果number_1 > number_2为true则返回number_1,为false则返回number_2
  • 三元表达式结果必须被使用
